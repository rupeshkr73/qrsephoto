FAQ

Common sawaal

Kya monthly subscription dena padega? ▼
Bilkul nahi. Sirf ek baar ₹499 setup fee deni hai. Uske baad koi monthly charge nahi. System lifetime use kar sakte ho.
Printer kaunsa chahiye? ▼
HP m1005, Brother 520w, Kyocera 2040dn, Epson L130, L8050, L18050 — sabhi common printers support hote hain. Auto-detect feature hai dashboard mein.
BG remove ke liye internet chahiye kya? ▼
Pehli baar page load hote waqt model download hota hai (ek baar). Uske baad sab kuch browser mein locally hota hai — koi API call nahi, bilkul free.
Print Agent .exe install karna hai kya? ▼
Haan, sirf ek baar install karna hai. Ye background mein chalega, System Tray mein rahega, PC restart pe auto-start hoga, aur future updates automatic aa jayenge.
Customer ka photo safe rahega kya? ▼
Print ke baad photo automatically delete ho jaata hai server se (Cloudinary auto-delete). Customer ki privacy poori tarah safe hai.
Price change karna ho toh? ▼
Shop Owner Dashboard se jab chahein price change kar sakte ho. Purana QR code bhi new price ke sath kaam karta hai — naya QR banana zaroori nahi.
